# Git
1. Названия файлов в решении и проектах лучше давать на английском языке. На сервере может не оказаться русской локализации.
2. Используйте конвенцию (правила) по наименованию, проектов, решений. Более подробно можно прочитать тут:
https://www.calabonga.net/blog/post/naming-for-projects-in-solution
